# El Umbral WoD 🦇
**Entorno eficaz de apoyo a narradores de juegos del universo de World of Darkness.**

Proyecto de Fin de Ciclo - Grado Superior en Desarrollo de Aplicaciones Web (DAW).

## 🚀 Tecnologías
* **Backend:** PHP 8.2+ / Laravel (API REST)
* **Frontend:** Angular 17+ / Tailwind CSS
* **Base de Datos:** MariaDB / MySQL
* **Real-time:** WebSockets (Laravel Reverb/Pusher)
* **Servidor:** Debian 12 / Nginx
